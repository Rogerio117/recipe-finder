# Recipe Finder 🍳

A recipe search app built with vanilla JavaScript 
and the Spoonacular API.

## Features
- Search recipes by name or ingredient
- Filter by diet type (vegan, gluten-free, etc.)
- View full recipe details and instructions

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Spoonacular API

1. Clone the repo
2. Add your Spoonacular API key in js/config.js
3. Open index.html with Live Server
